📋 说明

模式: Host to Host (H2H) - Host内存之间的直接通信

协议: COMM_PROTOCOL_UBC_CTP (华为统一总线CTP协议)

执行位置: Host CPU,无需Device Kernel

产品支持: Ascend 950PR / 950DT
